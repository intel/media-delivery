measure(1)
==========

NAME
----
measure - run media delivery measurements

SYNOPSIS
--------
[verse]
'measure' [<options>] [<command> [<args>]]

DESCRIPTION
-----------
`measure` runs media delivery quality or performance measure.

COMMANDS
-------
perf::
	Run performance measure

quality::
	Run quality measure
